# MakePhotos Skill

AI agent skill for [MakePhotos](https://makephotos.ai) — generate studio-quality product photos with AI.

Published on [LarryBrain](https://www.larrybrain.com), the skill marketplace for AI coding agents.

## What This Is

This repo contains a `SKILL.md` that teaches AI agents how to use the MakePhotos API to:

- Generate professional product photos (studio, Amazon, luxury, lifestyle)
- Remove image backgrounds
- Upload local files for processing
- Check credit balances

## For AI Agents

The `SKILL.md` file is what your agent reads. It includes setup instructions, full SDK and REST API usage examples, parameter references, and error handling.

## For Humans

If you want to use MakePhotos directly:

1. Sign up at [makephotos.ai](https://makephotos.ai/signup)
2. Install the SDK: `npm install makephotos`
3. See the [SDK documentation](https://github.com/makephotos/sdk)
